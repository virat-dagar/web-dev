## Project Overview

This project is a **one-page personal portfolio website** built using **semantic HTML**.
It presents basic personal information, academic projects, technical skills, and a contact form, following proper structure, accessibility, and validation standards.

The project is designed to be **clean, readable, and easy to style later using CSS**, as required in the lab instructions.

## Sections Included

1. **Header & Navigation**

   * Contains the student name and internal navigation links.
   * Includes a “Skip to main content” link for accessibility.

2. **Hero Section**

   * Welcome message and short introduction.

3. **About Section**

   * Profile image with proper alt text.
   * Short bio describing academic background.

4. **Projects Section**

   * Pattern Printer
   * Portfolio Website
   * Calculator

5. **Skills Section**

   * Displayed using a table.
   * Lists technical skills with learning levels.

6. **Contact Section**

   * Functional form with Name, Email, and Message fields.
   * Uses labels, placeholders, and required attributes.

7. **Footer**

   * Contains copyright and academic details.
